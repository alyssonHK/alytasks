# Gerenciador de Tarefas e Notas com IA

Um aplicativo web para gerenciar tarefas e notas com integração de IA, desenvolvido com Vite, TypeScript e Firebase.

## Funcionalidades

- Gerenciamento de tarefas com sub-tarefas
- Sistema de notas
- Canvas interativo para visualização de tarefas
- Integração com IA para análise
- Autenticação com Firebase
- Modo offline
- Interface responsiva

## Como usar

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Configure as variáveis de ambiente
4. Execute em desenvolvimento: `npm run dev`
5. Build para produção: `npm run build:gh-pages`
